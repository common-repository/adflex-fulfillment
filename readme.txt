Plugin support sellers POD. With Adflex Fulfillment Plugin, You can sync orders from your wordpress site to Adflex Fulfillment system. When we receive your order on our Adflex Fulfillment system, we will process it (producing, packing, shipping)

== Description ==

Feature:
* Sync order from your wordpress site to Adflex Fulfillment
* Submit order from Wordpress site to Adflex Fulfillment
* Auto receive tracking code and Fulfilled status from Adflex fulfillment

How to use:
For more information, please read document https://fulfillment.adflex.vn/how-to-use/

Privacy policy:
Please read and accept our privacy policy
https://fulfillment.adflex.vn/privacy-policy/

== Install ==
Download plugin and upload to wp-content/plugins. After that, Active plugin
